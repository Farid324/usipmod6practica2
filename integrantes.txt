Practica 2 Vue3

Nombre(s) de integrante(s):
- Farid Arancibia

Temática:
Tienda de Ropa - Gestión de Prendas y Categorías

Pasos para iniciar el proyecto:
1) Instalar dependencias:
   npm install

2) Instalar axios (si aún no está):
   npm install axios

3) Ejecutar frontend:
   npm run serve

4) Ejecutar backend (JSON Server):
   npx json-server --watch db/db.json --port 3000
